## 简介

***(InChat)Iot Netty Chat Cloud***

**核心项目[InChat](https://github.com/UncleCatMySelf/InChat)**


一个轻量级、高效、分布式的异步通信框架, 支持聊天和物联网, 您可以使用它来快速构建具有后台的聊天服务器, 并快速自定义自己的通信 api, 包括具有不同的通讯可以支持的协议。

本项目是其分布式组件，Cloud组件，统一结合数据处理与HTTP业务下发与查询
